Tiempo Real

Resultados se actualizan mientras escribes

Acceso a la Búsqueda:

  • Widget Principal: Click derecho → "🔍 Buscar CheatSheets"
  • TagManager: Pestaña CheatSheets → Barra de búsqueda
  • Menú Dial: Click derecho en el centro → "🔍 Buscar"

📄 Formato de CheatSheet

El formato es simple e intuitivo:

  • comando/objeto - descripción
  •   ejemplo práctico con indentación
  • Línea vacía para separar comandos

Ejemplo:

git status - Ver estado del repositorio
  git status --short

find . -name - Buscar archivos por nombre
  find . -name "*.py"

docker ps - Listar contenedores activos
  docker ps -a --format "table {{.Names}}\t{{.Status}}"
